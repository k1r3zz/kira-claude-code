# kira-cluade-code

Kira 的 Claude Code 技能集合，用于增强 AI Agent 的能力。

## 目录结构

```
kira-cluade-code/
├── README.md
├── config.yaml
└── skills/
    └── business-tracker/    # 业务追踪技能
```

## Skills

| 技能 | 说明 |
|------|------|
| [business-tracker](skills/business-tracker/) | 项目业务模块追踪、扫描与查询 |

## 使用方式

将 `skills/` 下的技能目录复制或软链接到 OpenClaw 的 skills 目录中即可使用。
